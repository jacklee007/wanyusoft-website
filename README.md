# WanyuSoft English Static Site
Static website for WanyuSoft (English version), built with Bootstrap 5.

## File Structure
- index.html: Homepage (with carousel & logo)
- about.html: About Us page
- our-services.html: Core services (AI Assistant, Website Development, Business System Development)
- our-cases.html: Successful cases (3 cases in English)
- pricing.html: Pricing plans (≤$2000 per person per month)
- contact.html: Contact information (no form, only direct contact)
- 404.html: 404 error page
- assets/: Static resources folder (for images)

## Quick Start
1. Clone or download all files to your local machine
2. Double-click `index.html` to open the site in browser
3. For server deployment: upload all files to your web server root directory (or Cloudflare Pages)
