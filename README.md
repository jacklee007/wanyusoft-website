# WanyuSoft English Static Site
Static website for WanyuSoft (English version), built with Bootstrap 5.

## File Structure
- index.html: Homepage
- about.html: About Us page
- our-services.html: Core services (AI Assistant, Website Development, Business System Development)
- pricing.html: Pricing plans
- contact.html: Contact information
- 404.html: 404 error page

## Quick Start
1. Clone or download all files to your local machine
2. Double-click `index.html` to open the site in browser
3. For server deployment: upload all files to your web server root directory
